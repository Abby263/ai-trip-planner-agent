# Roadmap

## Phase 1

- Monorepo, FastAPI, Next.js, Docker Compose, mock mode, health endpoint.

## Phase 2

- LangGraph orchestration, mock providers, itinerary options, budget, validation, final structured response.

## Phase 3

- Premium planner UI, progress, itinerary cards, map, budget, booking links, refinement.

## Phase 4

- Google Places, Google Routes, geocoding, route polylines, current-location planning.

## Phase 5

- Amadeus, Duffel, Ticketmaster, weather, currency, provider caching.

## Phase 6

- Session memory, long-term preference memory, pgvector preference retrieval, user permission flows.

## Phase 7

- Auth, rate limits, Sentry, LangSmith, OpenTelemetry, background workers, production deployment.
